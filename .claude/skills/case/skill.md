# /case - 案件進度查詢

## 描述
快速查詢特定案件的處理進度，包括轉錄狀態和 MEDDIC 分析結果。

## 使用方式
```
/case 202601-IC918    # 完整案件編號
/case IC918           # 簡短格式
/case 918             # 只輸入數字
```

## 執行流程

**直接執行腳本，不需要額外步驟：**

```bash
bun run scripts/case-lookup.ts <案件編號>
```

## 注意事項

1. **案件編號格式** - 支援多種輸入格式：
   - 完整格式：`202601-IC918`
   - 簡短格式：`IC918`
   - 數字格式：`918`

2. **查詢範圍** - 目前查詢最近 100 筆案件

3. **即時性** - 資料直接從 production API 取得，無快取延遲
